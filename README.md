# Tiger Island v1.0

Tiger Island is a two-player tile-placement game assigned as a project by Dave Small. 

The following concepts are currently in development:

* Game board
  * Tile
    * Hex
      * ~~Level~~
      * ~~Terrain~~
        * Uninhabitable 
        * Habitable 
      * ~~Edge~~

* Piece
  * Villager
  * Totoro

* Player
  * Settlements
  * Score
  * Turn
    * Action
      * Draw
      * Place tile
      * Place piece
      * Place villager
        * Expand
        * Found settlement 
      * Place Totoro

* End Game
  * Run out of pieces
  * Disqualified
